Educational Explanation

What is Offline Password Cracking? It is a simulation where an attacker tries many guessed passwords against a stolen hash offline. The attack is expensive and depends on how the hash was generated.

Why bcrypt is safer than MD5? bcrypt is intentionally slow and includes salt, so guessing many passwords is much more time-consuming than with MD5.

Educational Warning

This page demonstrates the concept using anonymized samples only. It never uses real leaked databases and never attacks real accounts.
